# Hobbies

---

![](/hobbies.jpeg)

I've been a CrossFitter for roughly 10 years now. I started when RedEye had an
office above the Brisbane CBD CrossFit gym. Between the performance tracking and
social element with coworkers, I had a blast and have continued to keep
it up despite moving around and the work from home evolution.

I run a home lab server for some home automation, game server hosting and network
RAID storage. It's a simple setup just running Portainer, but it lets me automate
some switches (fairy lights, heaters, Christmas trees) in the house and host
servers for my friends to play on. It's also handy to have a backup for my data.
I tried the Raspberry Pi cluster route and just found that for running game
servers they fell too short. So it became a single PC running a set of docker
containers. In hindsight, the simplicity at home is best.

In my downtime, I like to game on PC. My favourite titles recently include
Helldivers 2, Valheim and Factorio. I've played on keyboard & mouse forever, 
and I'm trying to learn to play with a Steam Deck, which is proving a challenge 
to learn to use a controller. It takes a lot to build up the muscle memory.
